## API Laravel


## Installation

Fazer o clone do projeto

Instalar as dependências
```bash
composer update
```
Modificar o arquivo .env com as credentials de banco

```
DB_HOST=localhost
DB_DATABASE=api
DB_USERNAME=Api
DB_PASSWORD=api

```

Rodar a migration para criação das tabelas

```bash

php artisan migrate

```

Rodar as Seeds


```bash

php artisan db:seed

```
